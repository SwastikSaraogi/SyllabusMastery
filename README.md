This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Babel](https://babeljs.io/) (or [oxc](https://oxc.rs) when used in [rolldown-vite](https://vite.dev/guide/rolldown)) for Fast Refresh
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/) for Fast Refresh

## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).

SyllabusMastery — Adaptive Learning & Assessment Engine

SyllabusMastery© is a client-side adaptive learning system that transforms raw syllabus content into a structured, mastery-driven assessment workflow. Built with React, the platform parses uploaded PDF or text-based syllabi into hierarchical units and topics using a rule-based parsing engine. Once structured, each topic becomes an assessable learning node powered by a deterministic adaptive algorithm that continuously adjusts difficulty based on individual performance.©

Unlike static quiz applications, SyllabusMastery tracks topic-level progress through a structured mastery model that includes attempts, best score, streak count, weighted mastery percentage, and spaced repetition scheduling. The system dynamically selects question difficulty (Easy, Medium, Hard) using performance trends rather than single-attempt scores. Mastery is updated using a weighted formula to prevent score inflation and to reflect consistent understanding over time.

The question engine supports multiple formats including MCQs, short answers, and long analytical responses. Evaluation logic combines correctness checks, keyword relevance detection, word-count heuristics, and difficulty weighting to generate structured feedback. Weak topics are automatically prioritized, and review intervals are scheduled based on score thresholds to reinforce long-term retention.
